## Pay Structure Reference

Based on the data, the driver pay formula appears to be:

| Component | Rate |
|-----------|------|
| Base Pay | $13 - $33 (varies by vendor/headcount) |
| Mileage Rate | $0.70/mile |
| Minimum Mileage Pay | $7.00 |
| Bonus | $10.00 (when qualified) |
| Tolls | Added when applicable |

**Formula**: `Total Driver Pay = Base Pay + max(Mileage × $0.70, $7.00) + Bonus + Tolls`
